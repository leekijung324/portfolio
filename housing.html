Key Drivers of Housing Prices: Understanding the Impact of Property Features on Sale Value

Ki Jung Lee

Executive Summary

This case study presents a comprehensive multiple regression analysis examining the factors that influence housing prices in Cincinnati. Through systematic data cleaning, exploratory analysis, and model development, we identified key predictors of residential property values. The analysis demonstrates that structural features—particularly square footage, number of bedrooms, and lot size—are the strongest drivers of home prices. The final regression model accounts for 43.7% of the variance in sale prices and provides valuable insights for homeowners, buyers, and real estate professionals seeking to understand residential property valuations.

1. Introduction: Description of Goals

Housing prices are affected by several factors, which include the size, quality, and desirability of a home. To understand the factors that influence housing values, this study aims to investigate the relationship between key characteristics of residential dwellings and their eventual sale prices. The analysis addresses two core research questions:

  1. How are the number of bedrooms, number of bathrooms, year built, square footage, lot size, and number of parking spots related to the sale price of a home?
  2. Which housing characteristics have the highest impact on sale price?

To analyze these questions, this study performs a multiple regression analysis using housing sales data. The findings have implications for homeowners seeking to optimize their property's value, home buyers evaluating pricing, and policymakers crafting effective housing market measures.

2. Data Collection and Cleaning

2.1 Data Source and Overview

The dataset comprises housing sales data from Cincinnati containing property characteristics and corresponding sale prices. The raw data includes 11 variables: Contributor, Address, Zipcode, Price, Date, Numbed (number of bedrooms), Numbath (number of bathrooms), Yearbuilt, Sqft (square footage), Lotsize (lot size in square feet), and Numparking (number of parking spots).

2.2 Data Cleaning Process

Data cleaning consists of two main steps: data preprocessing and outlier removal.

Step 1: Preprocessing

  • Column standardization: Column names were created as a names vector and passed to read_csv() to ensure proper column labels during import.
  • Data type conversion: The Numbath, Lotsize, and Numparking columns were converted to numeric data types.
  • Duplicate removal: Duplicate rows were removed by standardizing the Address column.
  • Missing value handling: Null values were removed by applying na.omit().

Step 2: Outlier Removal

Following preprocessing, filters were applied to remove observations with non-sensical or extreme values:

  • Age is restricted to be greater than or equal to 0 (no future ages; created as 2024 - Yearbuilt)
  • Square footage must be positive
  • Square footage cannot exceed lot size
  • Lot size cannot exceed 100,000 square feet
  • Price must be greater than $10,000

The final clean dataset contained 911 observations with 7 variables: Price, Numbed, Numbath, Age, Sqft, Lotsize, and Numparking.

2.3 R Code for Data Cleaning

library(tidyverse)

# Load data with proper column names
names <- c("Contributor", "Address", "Zipcode", "Price", "Date", "Numbed",
"Numbath", "Yearbuilt", "Sqft", "Lotsize", "Numparking")
housing <- read_csv("Datasets/housing.csv", col_names=names, skip=1)

#Convert data types
housing <- housing %>%
    mutate (Numbath = as.numeric (Numbath)) %>%
    mutate (Lotsize = as.numeric(Lotsize)) %>%
    mutate (Numparking = as.numeric (Numparking))

# Remove duplicates
housing$Address <- gsub(",","", housing$Address)
housing$Address <- gsub("\\s+"," ", housing$Address)
housing$Address <- tolower(housing$Address)
housing <- housing[!duplicated (housing$Address),]

# Remove null values
housingtb <- na.omit(housing)

# Create Age variable and apply outlier filters
housingtb <- housingtb %>%
    mutate(Age=(2024-Yearbuilt)) %>%
    filter(Age >= 0, Sqft>0, Sqft<=Lotsize, Lotsize <=100000, Price >=10000) %>%
    select(Price, Numbed, Numbath, Age, Sqft, Lotsize, Numparking)

4.4 Transformed Models for Assumption Checking

To address violations of homoscedasticity and normality, the skewed Price variable was log-transformed.

# Model 3: Log-transformed Price with all variables including Numbath

summary(model3)

# Model 4: Log-transformed Price without Numbath

5. Residual Analysis and Model Diagnostics

Model adequacy checking ensures the assumptions of the linear regression model (LINE) are reasonably satisfied. Residual plots for the initial models (Model 1 & 2) showed a funnel shape, indicating unequal error variance (heteroscedasticity). Q-Q plots also showed heavy-tailed distributions, violating the normality assumption. The log transformation of Price (in Models 3 & 4) helped to address these issues.

6. Model Validation

6.1 Cross-Validation Methodology

Five-fold cross-validation ($m=5$) was performed to compare the two transformed models (Model 3 with Numbath and Model 4 without Numbath) using PRESS (Prediction Error Sum of Squares), MSPE (Mean Squared Prediction Error), and Prediction R-squared.

6.2 Cross-Validation Results

Model                       PRESS     MSPE   Prediction $R^2$
Model 3 (with Numbath)    206.89   0.2271          0.5199
Model 4 (without Numbath) 248.34   0.2726          0.4237

6.3 Model Comparison and Selection

Based on the cross-validation metrics, Model 3 (with Numbath) demonstrates superior predictive performance. However, Model 4 is recommended as the final model for practical application because it is more parsimonious (fewer predictors) and provides more natural interpretation without the multicollinearity complications.

7. Key Findings and Insights

7.1 Most Influential Predictors

  1. Square Footage (Sqft): The most powerful predictor. Each additional square foot increases price by approximately $123.53.
  2. Number of Bedrooms (Numbed): The second most influential. Each additional bedroom increases price by approximately $119,937.
  3. Lot Size (Lotsize): A significant predictor. Each additional square foot of lot size increases price by approximately $1.57.
  4. Age: Statistically significant with a negative effect. Each additional year of age decreases price by approximately $3,092.

7.2 Model Performance

The final model (Model 4) explains 46.41% of the variance in housing prices. The remaining 53.59% is attributable to other factors not captured, such as neighborhood, condition, and amenities.

7.3 Unexpected Findings

The number of bathrooms (Numbath) lacked independent significance. This counterintuitive finding is due to high multicollinearity with the number of bedrooms—bathrooms tend to increase proportionally with bedrooms, reducing the independent contribution of the bathroom variable.

8. Recommendations

  • For Real Estate Professionals: Emphasize properties' square footage prominently in marketing materials, as this is the strongest value driver.
  • For Homeowners: Home improvement investments should prioritize increases in square footage or functional bedroom space, which demonstrate the strongest market impact.
  • For Investors: Focus on properties where square footage relative to price is attractive. Older properties with deferred maintenance may offer value opportunities.

9. Limitations and Future Work

9.1 Study Limitations

  • Sample bias: The sampling process appeared biased toward recently sold properties from Zillow.com.
  • Geographic specificity: Results are specific to Cincinnati and cannot be generalized.
  • Missing variables: The model excludes important qualitative factors like property condition, neighborhood characteristics, and school district ratings.

9.2 Potential Model Improvements

Future work could incorporate additional predictors (e.g., neighborhood data, school ratings), explore advanced modeling techniques (e.g., random forests), and implement spatial analysis to improve predictive power.

10. Conclusion

This regression analysis confirms that structural features are the primary drivers of housing prices in Cincinnati, with square footage, number of bedrooms, and lot size exhibiting the strongest relationships with sale prices. The final recommended model (Model 4 with log-transformed price) accounts for 46.41% of the variance in housing prices. These findings provide a practical, interpretable framework for understanding Cincinnati housing prices and supporting data-driven decision-making in the real estate market.

Appendix: Technical Specifications

Statistical Models

Multiple Linear Regression (Baseline):

$$ \hat{Price} = \beta_0 + \beta_1 \cdot Numbed + \beta_2 \cdot Age + \beta_3 \cdot Sqft + \beta_4 \cdot Lotsize + \beta_5 \cdot Numparking + \epsilon $$

Log-Transformed Regression (Final Model):

$$ \ln(\hat{Price}) = \beta_0 + \beta_1 \cdot Numbed + \beta_2 \cdot Age + \beta_3 \cdot Sqft + \beta_4 \cdot Lotsize + \beta_5 \cdot Numparking + \epsilon $$

Cross-Validation Formula:

$$ MSPE = \frac{1}{n}\sum_{i=1}^{n} (y_i-\hat{y}_{(-i)})^2 $$
